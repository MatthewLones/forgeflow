# Claims Precedents

## Overview

This skill provides access to precedent claims data for evaluating insurance claim amounts. Use it to find comparable settlements, identify patterns in claim outcomes, and support fair claim valuations.

## How to Use This Skill

When researching comparable claims:

1. Identify the claim type and category
2. Search for precedents with similar damage types and amounts
3. Note the settlement ranges for comparable claims
4. Consider regional and temporal adjustments
5. Document the precedent basis for recommended settlement amounts

## Precedent Categories

- **Water damage**: Pipe bursts, flooding, roof leaks
- **Fire damage**: Structure fires, smoke damage, electrical fires
- **Wind/hail**: Storm damage, tornado, hurricane
- **Theft/vandalism**: Break-in, property destruction
- **Liability claims**: Slip and fall, property damage to others
